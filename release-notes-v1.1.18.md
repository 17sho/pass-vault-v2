# Pass Vault V2 v1.1.18

## 中文

- 修复“修改登录名”弹窗在 iPhone 和窄屏浏览器中的标签、输入框重叠。
- 三个字段统一为单列布局，控件与操作按钮均满足至少 44px 触控高度。
- 当前账户名保持只读灰色样式；密码显隐及修改用户名行为/API 不变。

## English

- Fixed overlapping labels and inputs in the Change Login Name dialog on iPhone and narrow browser viewports.
- All three field groups now use a single-column layout with controls and actions at least 44px high.
- The current username remains read-only and visually muted; password-toggle and username-change behavior/API are unchanged.
